Pong - The Beginning

To begin, we will be doing the common functions that are usually called in setup as follows:

Next, we need to create global variables for what will be tracked. This will include:

  • Player's position
  • Bot's position
  • Ball's position
  • Ball's movement direction

We must initialize these outside all functions. I am going to include it in the new function initialValues() because doing this makes it easier to play a game again without exiting it.

You may notice that PVectors are being used, rather than ints or floats. Don't overthink these, as they simply store two values (x, y) so, for example, ballPos coordinates will be (ballPos.x, ballPos.y). Through using vectors, later on, we can simply just add the acceleration of the ball to the position.

Secondly, I have used the keywords width and height. These simply store the value of width and height which was set in the size() function. It is good to use these where applicable so resizing the window will be easier.
