Prerequisites

  • Before attempting this, you should have some programming experience
  • This guide is meant for people who know programming, but struggle to even do anything with their knowledge.
  • Simplistic games can be made with almost no programming experience, but the more the better
  • The only languages used will be Java and JavaScript

Making a game - picking your type

In this guide, you will have three main options to create a game. These will include using:

  • Processing
  • P5.js
  • Processing for Android

These all stem from one main source, Processing


Now if you have never heard of any of these, here is what each will produce:

  • Processing
    • This will run on your desktop
    • It uses Java
    • Therefore, it would be used to create a game that runs on your computer
  • P5.js
    • This will run in a browser
    • It uses JavaScript
    • This is what I recommend the most
    • It allows for easy sharing with friends
  • Processing for Android
    • This can be used to create an Android App
    • It uses Java
    • I would also highly recommend this
    • It is very easy to create, and even upload to the Google Play Store
